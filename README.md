# Email logs demo

Requires node 10 or above 

## Start app in 2 steps

### 1. Install dependencies
```
npm install
or
yarn install
```

### 2. Start app (Choose one of the two options)
-  With a local api
```
npm run start:svcfin
or
yarn run start:svcfin
```
- With a mock data server
```
npm start
or
yarn start
```
